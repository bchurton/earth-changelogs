# Earth Changelogs

### Want to contribute?

Create a pull request with your changes.
Changes must be in the format of:

##### Filename
x.x.x.json

##### Contents
```json
{
  "version": "x.x.x",
  "added": [
    "Added this",
    "Added that"
  ],
    "removed": [
        "Removed this",
        "Removed that"
    ]
}
```
